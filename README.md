# Pesaway Assistant
# Overview
PwBot is a web-based chatbot designed to provide quick, accurate, and professional information about Pesaway’s services. It helps users learn about payment options, API integrations, merchant benefits, and more through an interactive interface.

#Features
* *Service Information – Answers queries about MROs, card payments, e-Retail, API integration, and support contacts.
* *Currency Conversion – Converts between currencies using live exchange rates.
* *Math Operations– Performs basic calculations with error handling.
* *Responsive UI – Clean, professional design in Pesaway’s brand colors (blue and orange).
* *Dynamic Conversation Flow – Guides users with predefined responses for common questions.

# Project Structure

## Getting Started
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Type a question in the input field and click "Send" or press Enter.

# Example Queries

* What is Pesaway?
* Tell me about e-Retail.
* How do I integrate via API?
